### 1️⃣ PDF 习题转 PPT 工具

**让课件制作效率提升 10 倍**

#### 🎯 工具定位
这是一款专为试卷讲评而设计的课件制作神器，解决了"从 PDF 习题册制作讲评课件"这一高频刚需场景。

#### 💭 痛点分析
老师们在讲评试卷时，常常需要：
- 打开 PDF 习题册或试卷
- 截图保存每道题目
- 逐一插入 PPT
- 调整图片大小和位置
- 重复以上步骤几十次……

一套试卷讲评课件制作下来，往往需要 1-2 小时，枯燥且低效。

#### ✨ 解决方案
**上传 → 框选 → 导出，只需 3 步，3 分钟完成课件制作！**

#### 🔧 核心功能
1. **PDF 上传与实时预览**
   - 支持拖拽上传或点击选择
   - 多页 PDF 实时渲染预览
   - 支持高清显示，题目清晰可见

2. **精准裁剪工具**
   - 鼠标框选即可精准裁剪题目区域
   - 支持调整裁剪框大小和位置
   - 实时预览裁剪效果

3. **幻灯片管理**
   - 自由拖拽调整幻灯片顺序
   - 多张图片可合并到一页
   - 一键删除不需要的幻灯片

4. **一键导出 PPTX**
   - 生成标准 PowerPoint 格式
   - 可直接用 WPS/Office 打开编辑
   - 图片自动适配幻灯片尺寸

5. **现代化界面**
   - 简洁直观的操作界面
   - 无需学习成本，上手即用
   - 响应式设计，各种屏幕尺寸完美适配

#### 📝 使用流程
```
Step 1 → 上传包含习题的 PDF 文件
Step 2 → 在预览页面用鼠标框选需要的题目
Step 3 → 调整幻灯片顺序，合并或删除

```

### 2️⃣ 背诵任务管理系统

**让背诵检查井然有序**

#### 🎯 工具定位
一款专为语言类学科设计的班级背诵任务管理工具，实现多班级、多任务、多学生的可视化管理。

#### 💭 痛点分析
语文、英语、历史等学科的老师经常面临：
- 管理多个班级的背诵任务
- 记录每位学生的完成情况
- 用纸质表格容易丢失，不便统计
- 无法快速查看整体进度
- 数据难以分享给家长

传统的纸质登记表既不环保，也不方便查询和统计。

#### ✨ 解决方案
**数字化管理 + 可视化呈现，让背诵检查变得清晰高效！**

#### 🔧 核心功能

1. **多班级管理**
   - 同时管理多个班级
   - 一键切换不同班级
   - 每个班级独立的学生名单和任务

2. **任务管理**
   - 快速创建背诵任务（如"Unit 1-3 单词"）
   - 任务复制功能，避免重复输入
   - 任务编辑和删除，灵活调整

3. **学生管理**
   - 批量导入学生名单（支持粘贴）
   - 单个添加学生
   - 学生快速搜索定位

4. **完成状态跟踪**
   - 点击学生卡片即可切换完成/未完成状态
   - 实时统计完成人数
   - 可视化进度条，一目了然

5. **数据导出**
   - 导出为图片（可分享到家长群）
   - 导出为 Excel 表格（便于存档）
   - 导出为 CSV 文件（通用格式）

6. **智能搜索**
   - 快速搜索学生姓名
   - 高亮显示搜索结果
   - 大班级也能快速定位

7. **移动端适配**
   - 手机完美显示
   - 随时随地检查记录
   - 课间时间也能快速更新

8. **数据持久化**
   - 数据自动保存到浏览器本地
   - 刷新页面不丢失
   - 保护隐私，数据不上传服务器

#### 📝 使用流程
```
Step 1 → 创建班级并批量导入学生名单
Step 2 → 添加背诵任务（如"Unit 1-3"）
Step 3 → 学生完成后点击卡片标记完成
Step 4 → 导出数据留存或分享家长
```

#### 🎓 适用学科
语文、英语、历史、政治、地理等需要背诵记忆的学科

### 3️⃣ 课堂错题统计系统

**精准定位全班薄弱知识点**

#### 🎯 工具定位
一款课堂实时数据收集与可视化分析工具，通过随机点名和错题统计，快速识别高频错题，优化讲评效率。

#### 💭 痛点分析
讲评练习或试卷时，老师常常困扰于：
- 不知道哪些题目错误率最高
- 凭感觉判断重点讲解内容
- 全班讲一遍，耗时长效果差
- 无法量化学生的薄弱点

传统方式只能抽几个学生提问，无法获得全面的错题数据。

#### ✨ 解决方案
**随机点名 + 实时统计 + 热度可视化 = 数据驱动的精准讲评！**

#### 🔧 核心功能

1. **智能随机点名**
   - 公平随机抽取学生
   - 支持设置抽取人数
   - 避免总是点同一批学生
   - 增加课堂互动趣味性

2. **学生名单管理**
   - 批量导入学生名单
   - 支持多个班级切换
   - 名单编辑和更新

3. **动态题号矩阵**
   - 自定义题目数量
   - 自动生成题号按钮
   - 支持选择题、填空题、解答题等

4. **错题数据收集**
   - 学生点击自己做错的题号
   - 系统实时记录统计
   - 支持取消错选

5. **热度可视化**
   - 错题热力图展示
   - 错误次数越多颜色越深
   - 一目了然识别高频错题

6. **高频错题聚焦**
   - 自动排序显示错误率最高的题目
   - 建议优先讲解内容
   - 节省讲评时间

7. **关联分析**
   - 分析同时做错的题目组合
   - 发现知识点关联性
   - 深度挖掘学习问题

8. **统计报告导出**
   - 导出为图片保存
   - 可用于教学反思
   - 可分享给备课组

9. **数据持久化**
   - 本地自动保存
   - 支持查看历史统计
   - 数据安全可靠

#### 📝 使用流程
```
Step 1 → 导入学生名单，生成题号矩阵
Step 2 → 随机点名抽取学生上黑板
Step 3 → 学生点击自己做错的题号
Step 4 → 查看错题热力图和导出统计
```

#### 🎓 适用场景
- 课后作业讲评
- 周测、月考试卷讲评
- 课堂练习实时反馈
- 错题本数据收集

#### 🎓 适用学科
全学科通用 - 尤其适合理科（数学、物理、化学等）客观题较多的学科
