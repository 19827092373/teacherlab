# 英语背诵任务管理系统

一个专为教师设计的在线英语背诵任务管理工具，支持多班级管理、学生完成状态跟踪、数据导出等功能。

## ✨ 功能特性

### 📚 多班级管理
- 支持创建多个班级（如：一年级1班、二年级2班等）
- 每个班级独立管理学生名单和任务
- 快速切换班级，数据完全隔离
- 班级重命名和删除功能

### 📝 任务管理
- 为每个班级创建多个背诵任务
- 任务内容自定义编辑
- 任务复制功能，快速创建相似任务
- 任务删除和重命名

### 👥 学生管理
- 批量导入学生名单（支持换行分隔）
- 学生完成状态一键切换
- 实时统计：总人数、已完成、未完成、完成率
- 可视化进度条显示完成情况
- 学生搜索功能，快速定位

### 🎯 批量操作
- 全选/取消全选/反选学生
- 批量标记完成/取消完成
- 支持多选操作（Ctrl/Cmd + 点击）

### 📊 数据导出
- 导出为图片（支持单列、多列、紧凑布局）
- 导出为Excel/CSV文件
- 导出内容包含统计信息

### 📱 响应式设计
- 完美适配桌面、平板、手机
- 手机横屏自动优化布局
- 触摸友好的交互设计

### 💾 数据存储
- 数据自动保存到浏览器本地存储
- 无需服务器，完全离线可用
- 数据安全可靠

## 🚀 快速开始

### 在线使用

**在线访问地址：** https://tool3.teacherlab.top

直接访问部署的网站即可使用，无需安装。

### 本地部署

1. 克隆或下载项目
```bash
git clone <repository-url>
```

2. 打开 `英语背诵管理.html` 文件
   - 直接在浏览器中打开即可使用
   - 或使用本地服务器（推荐）

3. 使用本地服务器（可选）
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需要安装 http-server)
npx http-server
```

然后访问 `http://localhost:8000`

## 📖 使用指南

### 创建班级

1. 点击顶部"➕ 添加班级"按钮
2. 输入班级名称（如：一年级1班）
3. 点击确认

### 导入学生名单

1. 点击"👥 导入名单"按钮
2. 在文本框中输入学生姓名，每行一个
3. 点击"确认导入"

### 创建任务

1. 在任务标签区域点击"新任务"按钮
2. 在文本框中输入任务内容
3. 任务会自动保存

### 标记学生完成状态

- **单个标记**：点击学生卡片即可切换完成/未完成状态
- **批量标记**：
  1. 使用批量操作工具栏选择学生
  2. 点击"批量标记完成"或"批量取消完成"

### 导出数据

- **导出图片**：点击"📷 导出图片"，选择布局后下载
- **导出Excel**：点击"📊 导出Excel"，自动下载CSV文件

### 切换班级

在顶部下拉框中选择要管理的班级，系统会自动切换。

## 🛠️ 技术栈

- **纯前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **无依赖**：不依赖任何框架，纯原生实现
- **存储**：localStorage（浏览器本地存储）
- **导出库**：html2canvas（图片导出）

## 🔒 数据安全

- 所有数据存储在浏览器本地（localStorage）
- 数据不会上传到任何服务器
- 清除浏览器缓存会清除数据，建议定期导出备份

## 📝 注意事项

1. **数据备份**：建议定期导出数据作为备份
2. **浏览器兼容**：建议使用现代浏览器（Chrome、Firefox、Safari、Edge）
3. **存储限制**：localStorage 有存储限制（通常5-10MB），如果数据量很大，建议分班级管理
4. **隐私模式**：某些浏览器的隐私模式可能会限制 localStorage 的使用
